The built-in admin dashboard is available at /admin in the main app.

For full CMS with GitHub integration and automatic deployment, connect this repository to Netlify and enable Netlify CMS or Sveltia CMS.

Open Admin Dashboard
Setup Instructions:

1. Push this project to a GitHub repository
2. Connect to Netlify for automatic deployment
3. In Netlify → Site Settings → Identity, enable Identity
4. Enable Git Gateway under Identity settings
5. Update admin/config.yml with your repo details
6. Access CMS at /admin and login with your Netlify Identity account
